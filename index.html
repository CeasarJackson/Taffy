Taffy

A REST Web Service Framework for ColdFusion and Railo

Download .zip Download .tar.gz View on GitHub

I created Taffy because I was not happy with the amount of code it took to create REST API's with ColdFusion. I wanted something terse, something that used convention over configuration, and something that didn't require writing a bunch of boilerplate code. Nothing existed that filled that void, so I wrote it.

Current Version

Version 1.1 was released on June 27th, 2012. (changelog)

Beta of Next Version!

Version 1.2 will be finalized soon, and is available for beta testing now. It is recommended that you not use it in production APIs yet. Access the beta as the "develop" branch in the github repository.
